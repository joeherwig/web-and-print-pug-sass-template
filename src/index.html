Foreword

This documentation template is intended to provide an easy base to start with writing documentation that should be published online and as PDF / for printing.

On providing this template and the approach to write the documentation i ensured that easy conversion to EPUBs like with Calibre is ensured by focussing on formatting based on CSS.
